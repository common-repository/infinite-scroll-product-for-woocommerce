Enhance your WooCommerce store with seamless AJAX Load More, Infinite Scroll, and AJAX Pagination features.

== Description ==
Transform your WooCommerce store with the ultimate product loading experience using Infinite Scroll Product For WooCommerce. This powerful plugin introduces features like AJAX Load More, Infinite Scroll, and AJAX Pagination, ensuring a smooth and uninterrupted shopping experience for your customers. Products load dynamically as users scroll or click to load more, eliminating the need for page reloads and keeping customers engaged.

== Installation ==
1. Upload the `infinite-scroll-product-for-wooCommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings Infinite Scrolling  Options menu.

== Frequently Asked Questions ==
= How do I configure the AJAX Load More feature? =
Navigate to Infinite Scrolling Options and enable AJAX Load More. Customize the button text and loader image as needed.

= Can I customize the loading animation? =
Yes, you can customize the loader image and text from the plugin settings.

= Is the plugin mobile-friendly? =
Absolutely! The plugin is optimized for mobile devices to provide a superior shopping experience on any device.

= Will this plugin affect my SEO? =
No, the plugin is designed to be SEO-friendly, ensuring your store remains optimized even with dynamic content loading.
